# Strategy Concept (策略概念)

## 1. 核心理念 (Core Philosophy)
*在此描述策略的主要思路，例如：趨勢跟踪、均值回歸、網格交易等。*

## 2. 進場邏輯 (Entry Logic)
* **Long Entry**:
  * 條件 1: ...
  * 條件 2: ...
* **Short Entry**:
  * 條件 1: ...
  * 條件 2: ...

## 3. 出場邏輯 (Exit Logic)
* **Take Profit (TP)**: 固定點數 / 指標出場 / ...
* **Stop Loss (SL)**: 固定點數 / 波段高低點 / ...

## 4. 資金與風險管理 (Risk Management)
* **Lot Size**: 固定手數 / 動態手數 (Risk %) / Martingale ...
* **Max Drawdown**: 最大回撤限制
* **Filters**: 時間過濾 / 新聞過濾 / 點差過濾
